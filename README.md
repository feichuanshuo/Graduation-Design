# Graduation-Design
基于爬虫技术的西安房地产发展影响因素数据库设计

## 一、项目简介

- 分析、识别西安市房地产发展的供需影响因素（宏观因素和微观因素）
- 设计爬虫软件获取房地产发展的影响因素数据
- 构建房地产发展的影响因素数据库
- 基于上述数据的人工智能应答系统设计
